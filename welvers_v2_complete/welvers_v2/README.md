# Welvers

Международная SEO-платформа для поиска, анализа, сравнения и выбора AI-инструментов и SaaS-сервисов.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
# Python
python -m http.server 8080

# Node.js (если установлен)
npx serve .
```

Затем откройте http://localhost:8080

## Структура

- `index.html` — главная страница (RU)
- `css/styles.css` — стили (dark premium theme)
- `js/main.js` — интерактивность (меню, ⌘K)
- `assets/` — изображения и ресурсы
